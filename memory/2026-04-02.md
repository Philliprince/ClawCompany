## 🔧 技术改进

### OpenCode 调用方式修正
**问题**: 使用 `sessions_spawn` 启动 OpenCode 会失败
**原因**: Telegram channel 不支持 thread binding

**解决方案**: 使用 `exec` + `pty:true` + `background:true`
```bash
exec({
  command: "cd ~/project && opencode run '任务'",
  pty: true,
  background: true,
  timeout: 3600,
  workdir: "/path/to/project"
})
```

**监控方式**:
```bash
process({ action: "log", sessionId: "session-id" })
```
